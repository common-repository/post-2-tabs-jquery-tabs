Your best categories in a simple tabs widget (Jquery UI Tabs).

== Description ==
Simple plugin that allows you to generate a tab container widget based on Jquery UI tabs. Each tab is a category with a number of posts customizable. 
It will display last posts of your favorite categories just put each category ID number separated by comma.

All other information (general, changelog, installation, upgrade, usage) you need about this plugin can be found here actually in italian: [Post2tabs readme](http://www.giuseppesurace.com/ "Sendit Readme").

== Installation ==

1. Upload `post2tabs dir` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to post 2 tabs settings page and edit your setting. Choose what content do you want display in each tab
4. Move post 2 tabs widget in your favorite sidebar place
